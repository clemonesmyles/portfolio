# DVA Digital Transformation - Project Images

## Image Placeholders

This directory contains placeholder images for the DVA case study.

### To Add Your Images:

1. **Hero Image** (`hero.jpg`)
   - Recommended size: 1600x900px (16:9 ratio)
   - Replace the current placeholder SVG with your project hero image
   - Use JPG or PNG format

2. **Additional Images** (optional)
   - Add supporting images as: `01.jpg`, `02.jpg`, `03.jpg`, etc.
   - These can show:
     - Service design artifacts
     - Workshop photos
     - User journey maps
     - Research findings visualization
     - Before/after comparisons
     - Stakeholder presentations

### Image Guidelines:

- Compress images before uploading
- Ensure images don't contain sensitive information
- Use consistent aspect ratios for gallery images
- Consider accessibility: images should support the narrative
